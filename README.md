# Dream Emotion Analysis using NLP & Machine Learning

Analyze over **39,000 dream journal entries** using Natural Language Processing (NLP) and Machine Learning to identify emotional patterns and predict dominant emotional states.

---

## Project Overview

Dream narratives often contain valuable emotional and psychological information but are highly unstructured, making them difficult to analyze.

This project develops an end-to-end data analysis and machine learning pipeline that transforms raw dream journal entries into meaningful features for emotion prediction. It combines traditional data preprocessing techniques with transformer-based NLP models to analyze textual dream descriptions alongside structured demographic and contextual information.

---

## Problem Statement

How can unstructured dream narratives be transformed into meaningful insights that help predict the dominant emotional state expressed within dreams?

The project addresses this challenge through data cleaning, feature engineering, NLP-based emotion extraction, exploratory data analysis, and predictive modeling.

---

## Dataset

- **39,000+ dream journal entries**
- **184 structured and unstructured attributes**
- Web-scraped dream journal dataset
- Combination of textual, demographic, contextual, and metadata features

---

## Key Features

- Cleaned and preprocessed large-scale textual data
- Engineered meaningful features from 184 fragmented attributes
- Performed exploratory data analysis and visualization
- Applied Natural Language Processing using NLTK
- Extracted emotions using Hugging Face Transformers
- Built predictive machine learning models
- Evaluated model performance using classification metrics
